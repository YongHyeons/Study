신라면
  1. 재료 신라면 물(550ml) 파 계란
  2. 물을 끓인다
  3. 면을 넣는다
  4. 분말 스프와 취향에 따라 건더기 스프를 넣는다
  5. 3~4분 정도 기다린다
  6. 저어서 맛있게 먹는다
짜파게티
  1. 재료 짜파게티 물(550ml)
  2. 물을 끓인다
  3. 면과 건더기 스프를 넣는다
  4. 면을 그릇에 건져낸다
  5. 남은 면수는 버리지 말고, 스프가 뭉치지 않게 풀릴 수 있을 정도로 그릇에 따른다
  6. 스프를 섞는다
  7. 별첨된 올리브유로 면을 버무린다
  8. 맛있게 먹는다
팔도비빔면
  1. 재료 팔도비빔면 물(550ml)
  2. 물을 끓인다
  3. 끓는 물 600ml에 면을 3분동안 익힌다
  4. 찬물에 면을 헹군다.
  5. 양념장 소스에 비벼 먹는다
